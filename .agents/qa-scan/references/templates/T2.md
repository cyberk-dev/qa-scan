# T2 — Issue Confidence Thấp

**Trigger:** `qa-issue-analyzer` returns `confidence < 0.5` OR test scenarios missing.

## Question payload

> Phân tích issue có độ tin cậy thấp ({confidence}%). Bạn muốn tiếp tục thế nào?

## Options

| id          | label (VI)                              | description                                                          |
|-------------|-----------------------------------------|----------------------------------------------------------------------|
| rewrite     | Chỉnh sửa kịch bản (Khuyến nghị)        | Cung cấp mô tả chi tiết hơn, analyzer sẽ phân tích lại               |
| continue    | Tiếp tục với kịch bản hiện tại          | Chấp nhận độ tin cậy thấp, có thể dẫn tới FAIL/PARTIAL                |
| abort       | Huỷ scan                                | Issue quá mơ hồ, nên test thủ công                                    |

## Diagnosis block

```markdown
## 📋 Phân Tích Issue (Confidence: {confidence}%)

**TÍNH NĂNG:** {feature_area}
**KỊCH BẢN TEST:**
1. {scenario_1}
2. {scenario_2}

**HÀNH VI MONG ĐỢI:** {expected_behavior}

⚠️ Độ tin cậy thấp — cần xác nhận kỹ.
```
