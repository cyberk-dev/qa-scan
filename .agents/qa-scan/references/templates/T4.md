# T4 — Env Bootstrap Fail

**Trigger:** `qa-env-bootstrap` (Step 0a) fail — missing secret / port conflict unresolved / services not starting / monorepo no manifest.

## Question payload

> Env bootstrap lỗi: `{error_type}`. Bạn muốn xử lý thế nào?

## Options

| id              | label (VI)                                  | description                                                                                |
|-----------------|---------------------------------------------|--------------------------------------------------------------------------------------------|
| fix_secrets     | Cung cấp secret / fix config (Khuyến nghị)  | Prompt nhập các biến thiếu, lưu vào `~/.qa-scan/secrets/{repo}.yaml`                       |
| kill_port       | Kill process chiếm port                     | Force kill `lsof -ti:{port}` — CẢNH BÁO: có thể phá dev session khác                        |
| continue_partial| Tiếp tục với PARTIAL                         | Skip env setup, test với server hiện có nếu 200 OK                                         |
| abort           | Huỷ scan                                    | Env không sẵn sàng, test sau                                                               |

## Diagnosis block

```markdown
## 🔧 Env Bootstrap Failed

**Loại lỗi:** {error_type}
- Missing secrets: {missing_secrets_list}
- Port conflict: `port {port}` đang bị `pid {owner_pid}` (cwd: `{owner_cwd}`) chiếm
- Service không start: `{service_name}` healthcheck timeout {timeout}s

**Đề Xuất Sửa:** {recommendation}
```
