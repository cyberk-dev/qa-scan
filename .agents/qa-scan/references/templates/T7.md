# T7 — Verifier Phát Hiện Gap Nghiêm Trọng

**Trigger:** `qa-coverage-verifier` / `qa-adversarial-verifier` finds coverage gap > 50% OR adversarial probe exposes critical vulnerability.

## Question payload

> Verifier phát hiện gap nghiêm trọng: `{gap_summary}`. Verdict đề xuất: `{suggested_verdict}`. Bạn muốn xử lý thế nào?

## Options

| id            | label (VI)                            | description                                                          |
|---------------|---------------------------------------|----------------------------------------------------------------------|
| add_tests     | Thêm test cho gap (Khuyến nghị)       | Regenerate test với scenarios bổ sung, re-run                        |
| accept_gap    | Ghi nhận gap + tiếp tục                | Verdict = PARTIAL, gap được note trong report                         |
| override_pass | Override verdict thành PASS            | ⚠️ Chấp nhận rủi ro, document lý do bắt buộc                          |
| abort         | Huỷ scan                              | Cần redesign test approach                                            |
