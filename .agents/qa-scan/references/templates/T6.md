# T6 — Test Run Fail 3×

**Trigger:** `qa-test-runner` fails 3 consecutive runs (timeout / crash / selector miss).

## Question payload

> Test run fail 3 lần: `{last_error}`. Bạn muốn xử lý thế nào?

## Options

| id              | label (VI)                              | description                                                          |
|-----------------|-----------------------------------------|----------------------------------------------------------------------|
| edit_trace      | Xem trace + edit test (Khuyến nghị)     | Mở Playwright trace viewer, sửa selector/assertion                   |
| extend_timeout  | Tăng timeout + retry                    | timeout 30s → 60s, retry 1 lần nữa                                    |
| report_fail     | Report FAIL với evidence hiện có        | Verdict = FAIL, submit báo cáo kèm video/trace                        |
| abort           | Huỷ scan                                | Pipeline không thể proceed                                            |
